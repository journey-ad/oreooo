## Keyboard shortcuts

| Key       | Description                        |
| --------- | ---------------------------------- |
| o / r     | Input 'O' or 'RE' |
| - / Space | Input space             |
| Enter     | Generate                
| Backspace | Back                        |

## Contributing a translation

The text in Oreooo is in Chinese by default. You can translate it in your language. Just edit your language pack in `src/locales` and  `pull requests` to me. Thanks!
